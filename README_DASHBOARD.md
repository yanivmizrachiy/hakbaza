# 🎯 לוח בקרה - פרויקטים וריפוזיטוריות

לוח בקרה מתקדם לניהול וצפייה בכל הפרויקטים והריפוזיטוריות ב-GitHub.

## 🌐 גישה לאתר

האתר זמין ב: **https://yanivmizrachiy.github.io/hakbaza/**

## ✨ תכונות

- 📊 סקירה מלאה של כל הפרויקטים
- 🔍 חיפוש וסינון מתקדמים
- 📈 גרפים וסטטיסטיקות
- ⭐ מערכת מועדפים
- 📝 הערות אישיות לכל פרויקט
- 🔀 מצב השוואה בין פרויקטים
- 🌙 Dark Mode
- 📱 PWA Support - ניתן להתקין כאפליקציה
- 🔄 עדכון אוטומטי מ-GitHub API
- 📥 ייצוא ל-JSON/CSV
- 🖨️ תצוגת הדפסה
- 📤 שיתוף

## 🚀 שימוש

פשוט פתח את `projects-dashboard.html` בדפדפן או גש לאתר החי.

## 📁 קבצים

- `projects-dashboard.html` - הדף הראשי
- `index.html` - דף מעבר אוטומטי
- `sw.js` - Service Worker ל-PWA
- `manifest.json` - קובץ Manifest ל-PWA

## 🔧 טכנולוגיות

- HTML5
- CSS3 (עם CSS Variables)
- JavaScript (ES6+)
- Chart.js
- GitHub API
- PWA (Service Worker)
